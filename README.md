## â„¹ï¸ Overview

The purpose of this project is to make a code that randomly generates 10 distinct non-playable characters (npcs). The user can input their own choices or randomly pick them. Each character is distinct and unique with randomized character attributes. Some characters may not qualify for certain attributes, depending on the user's input. Each character has a name, a fight status, an occupation, shillings, and a societal status. This program uses basic progamming structures.
